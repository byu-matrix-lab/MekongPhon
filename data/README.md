# data/

## Subdirectories

- `lexicons/` — complete word-to-IPA lexicons for Khmer and Lao
- `opus_download/` — shell scripts to download OPUS parallel text corpora
- `sampled/` — 500-sentence samples used for qualitative review
- `splits/` — train and eval splits of the OPUS corpus after conversion

## Usage

See `../pipeline/` for instructions on how the lexicons and corpus splits are built from source data.
